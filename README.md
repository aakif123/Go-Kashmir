# Go Kashmir 🏔️

A full-stack tourism website for Kashmir built as a personal portfolio project.

## Tech Stack
- **Frontend:** Next.js 15 (App Router) + TypeScript
- **Styling:** Tailwind CSS v3
- **Backend:** Supabase (PostgreSQL + Auth + Storage)
- **Animations:** Framer Motion
- **Deployment:** Vercel

## Features
- 🏠 Beautiful home page with hero carousel and destination cards
- 🗺️ Destinations & Accommodations pages with search and filters
- 🔐 Authentication with email verification (modal with 3D flip animation)
- 🛡️ Role-based access (Admin / User)
- ⚙️ Admin panel with full CRUD for destinations and accommodations
- 📱 Fully responsive and mobile-friendly
- 📬 Contact form connected to Supabase

## Getting Started
1. Clone the repo
2. Run `npm install`
3. Create `.env.local` with your Supabase credentials
4. Run `npm run dev`
